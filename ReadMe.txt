файлы calc_*** - функции для решения д. у.

файлы geom_*** - функции создания и работы с геом. моделями.

cut - функция срезания материала.

main_*** - программы для моделирования процесса.

main_const - программа для моделирования с силами постоянными на шаге
по времени.

main_lin - программа для моделирования с силами линейными на шаге
по времени (без итерационного уточнения).

main_iter - программа для моделирования с силами линейными на шаге
по времени (с итерационным уточнением).

main_iter_DOF1 - то же, но только с одной степенью свободы.

main_iter_puankaracii - функция, возвращающая полученные в результате моделирования
динамические перемещения, которые потом используются для построения пуанкараций.

main_iter_puankaracii_8 - то же, но с глубиной рез-я 8 подач.

main_iter_puankaracii_1DOF - то же, но с одной степенью свободы.

main_iter_puankaracii_1DOF_8 - то же, но с глубиной рез-я 8 подач.

main_iter_tech_* - получение шероховатости для технологической части
в зависимости от *.

puank - основная программа для построения пуанкараций.

tech - программа для построения графиков в технологической части.
